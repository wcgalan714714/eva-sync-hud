# EVA Sync HUD

Meta Ray-Ban Display web app with Neon Genesis Evangelion UI aesthetic.

## Features
- Live Sync Gauge with dynamic transitions
- Vital signs tracking (Protein = LCL Conc., etc.)
- LocalStorage persistence
- Gesture simulation (Enter/Space = +10g protein)

## Testing
- Open index.html in browser
- Press Enter/Space to log protein
- Refresh to see persistence

## Deploy
Deploy on Vercel as a static site.
